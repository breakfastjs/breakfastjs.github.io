BreakfastJS is a Saturday social breakfast club for people learning to code.

1. Eat breakfast

+ People learn best in a social environment and on a full tummy. + Gives people a chance to meet & get to know each other. + BreakfastJS Club breakfasts should cater for different diets. Not everyone wants a fry up, so there will always be at least a vegetarian option. + Why is the breakfast bit important? + What meal of the day is better than breakfast?

2. Learn JavaScript

+ Programming language of the web + Great for beginners + No fancy software to install, you just a laptop with a web browser.

3. Meet New People

+ BreakfastJS uses pair programming to help you learn. + Clubs are a welcoming environment for people of all ages, backgrounds, genders and experience levels.

4. Become a Tech Wizard

+ Pro developer and want to learn how to coach + Just starting out and curious about the basics + Learning for a while and want to learn with others + BreakfastJS is for you.
